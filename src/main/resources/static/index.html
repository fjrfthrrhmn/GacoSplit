Langsung ke konten utama

Makan Bareng Gacoan,
Bayar Jadi Gampang

GacoSplit bantu kamu dan teman-teman hitung tagihan split bill secara otomatis—gak perlu pusing ngitung manual, langsung tahu masing-masing harus bayar berapa.

Mulai Sesi Baru
01

Input Peserta

Tambah nama setiap orang yang ikut makan bareng—minimal dua, maksimal sepuluh. Tiap peserta punya catatan pesanan sendiri, jadi gak bakal tercampur.

02

Tambah Item

Pilih menu dari daftar Gacoan atau input manual. Item shared otomatis dibagi rata ke semua peserta.

03

Hasil Otomatis

GacoSplit langsung hitung tagihan tiap orang—total personal ditambah bagian shared item. Hasil siap disalin atau di-screenshot. Adil, transparan, tanpa ribet.
