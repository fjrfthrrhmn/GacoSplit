# static/

Folder berisi aset statis untuk frontend.

- `css/input.css` — entry point Tailwind CSS dengan custom properties
- `css/output.css` — output build Tailwind
- `js/` — JavaScript aplikasi

Build CSS: `npx tailwindcss -i ./src/main/resources/static/css/input.css -o ./src/main/resources/static/css/output.css`
