# CSS — Tampilan dan Gaya Aplikasi

Folder ini berisi file CSS yang ngatur **tampilan visual** aplikasi GacoSplit.
Mulai dari warna, bentuk tombol, ukuran font, sampai layout halaman — semuanya diatur dari sini.

## 📌 Tujuan

- Menyediakan **design tokens** (warna, font, ukuran) yang dipake di seluruh halaman
- Definisikan **komponen reusable** kayak tombol, card, badge, footer — jadi gak perlu nulis ulang CSS tiap bikin elemen
- Bikin tampilan konsisten antara halaman `index.html` dan `start.html`
- Pake **CSS variables** (`--primary`, `--secondary`, dll) biar gampang ganti tema

## 📎 Related

- `/resources/templates` — HTML yang pake class dari CSS ini
- `DESIGN.md` (root project) — dokumen desain yang jadi acuan warna dan layout
