## 🖲️ CTA (Call to Action)

CTA adalah "next step" buat user. Gak boleh boring ato terlalu formal.

| Context          | Button Copy                                | Notes                                  |
| :--------------- | :----------------------------------------- | :------------------------------------- |
| Hero utama       | **Mulai Hitung, Yuk!**                     | Ajak langsung, ga pake lama            |
| Akhir flow       | **Hitung Lagi**                            | Reset sesi, wording positif            |
| Halaman start    | **Bikin Session Baru**                     | Jelas, trigger aksi                    |
| Konfirmasi hapus | **Yakin Mau Hapus?** + **[Hapus] [Batal]** | Konfirmasi dulu biar aman              |
| Copy hasil       | **Salin Hasil**                            | "Copy" diganti "Salin" biar lebih Indo |
| Tambah orang     | **+ Tambah Bestie**                        | Pake "bestie" biar relatable           |
| Tambah item      | **+ Tambah Makanan**                       | Gak pake "item", terlalu teknis        |
| Selesai          | **Udah, Gaskeun!**                         | Vibe anak gaul, trigger action         |

### CTA Don'ts

- ❌ "Submit" — jangan, terlalu formal
- ❌ "Proceed" — sok inggris banget
- ❌ "Execute Calculation" — literally jangan, ini bukannya app kantoran
- ❌ "Add Participant" — "tambah orang" aja cukup

## 📝 Form & Input Tagihan

Semua microcopy yang berhubungan sama form input. Ini bagian paling krusial karena user paling lama interaksi di sini.

### Form Labels

| Field            | Label              | Placeholder                | Notes                        |
| :--------------- | :----------------- | :------------------------- | :--------------------------- |
| Nama orang       | **Nama Bestie**    | "Contoh: Fajar"            | Gak pake "participant"       |
| Nama menu        | **Makanan**        | "Cth: Mi Gacoan Lv3"       | Pake singkatan yang familiar |
| Harga            | **Berapa Duid?**   | "Rp 18.000"                | Pake "duid" biar santai      |
| Quantity         | **Berapa Porsi**   | "1"                        |                              |
| Nama custom item | **Nama Makanan**   | "Cth: Dimsum 2 porsi beda" |                              |
| Budget / max     | **Budget Berapa?** | "Maksimal Rp..." (future)  |                              |

### Validation & Error Messages

| Scenario          | Error Copy                                                          | Tone        |
| :---------------- | :------------------------------------------------------------------ | :---------- |
| Nama kosong       | **Nama orangnya diisi dulu ya, gak boleh kosong.**                  | Tegur halus |
| Nama dobel        | **Udah ada yang pake nama ini, ganti dikit lah.**                   | Santai      |
| Harga bukan angka | **Coba cek lagi, harganya harus angka doang ya.**                   | Bantu       |
| Harga 0           | **Makanannya gratisan? Isi harga yang bener dong.**                 | Bercanda    |
| Quantity < 1      | **Pesennya minimal 1 porsi lah ya.**                                | Ngingetin   |
| Quantity > 99     | **Lu mau borong? Maksimal 99 aja dulu ya.**                         | Lucu        |
| Kurang 2 orang    | **Minimal 2 orang dulu ya baru bisa mulai.**                        | Informasi   |
| Belum ada pesanan | **Pesenannya diisi dulu dong, mau ngitung apa?**                    | Humor       |
| Shared item TAPI  | **Ini dibagi rata ya ke semua orang di session. Good luck bestie!** |             |

### Success & Feedback

| Scenario                | Toast/Message                                       | Notes                   |
| :---------------------- | :-------------------------------------------------- | :---------------------- |
| Orang berhasil ditambah | **Udah, [Nama] masuk!**                             | Langsung muncul di list |
| Menu berhasil ditambah  | **Mantap, [Menu] udah dicatet!**                    | Plus animasi centang    |
| Item dihapus            | **[Item] udah dicopot, aman.**                      | Konfirmasi berhasil     |
| Hasil berhasil di-copy  | **Udah ke-copy nih! Tinggal paste ke grup.**        | + toast                 |
| Ada error               | **Eh error nih. Coba refresh dulu atau ulangi ya.** | Gak usah teknis         |

### Empty State

| Section                     | Copy                                                       |
| :-------------------------- | :--------------------------------------------------------- |
| Belum ada orang             | **Belum ada orang nih. Klik "Tambah Bestie" dulu ya!**     |
| Belum ada pesanan per orang | **Si [Nama] belum pesen apa-apa. Kasian, coba tambahin.**  |
| Belum ada shared item       | **Makan bareng? Bisa tambah item yang dibagi rata nanti.** |
| Hasil kosong                | **Belum ada yang diitung. Isi dulu pesanannya ya.**        |

### Loading & Waiting

| Scenario     | Copy                                      |
| :----------- | :---------------------------------------- |
| Lagi loading | **Bentar ya, lagi gua proses...**         |
| Lagi ngitung | **Mikir bentar... banyak nih pesenannya** |
| Sukses load  | **Udah siap nih!**                        |

## ⚠️ Edge Cases & Error Pages

| Scenario        | Headline                        | Description                                              | Action                 |
| :-------------- | :------------------------------ | :------------------------------------------------------- | :--------------------- |
| 404             | **Waduh, halaman ini gak ada.** | Mungkin udah kedaluwarsa ato linknya salah.              | **Balik ke Home**      |
| 500             | **Maap, gua error.**            | Ada yang salah di server. Bisa coba refresh dulu.        | **Refresh**            |
| Session expired | **Session lu udah expired.**    | Mungkin udah direstart atau kehapus. Buat aja yang baru. | **Bikin Session Baru** |
| Network offline | **Lu lagi offline nih.**        | Cek koneksi internet dulu ya baru cobain lagi.           | **Refresh**            |
| Maintenance     | **Gua lagi diservis bentar.**   | Biasanya selesai beberapa menit lagi. Sabar ya bestie.   | —                      |
